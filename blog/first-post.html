Welcome to My Capstone Journey

Author: Walter Melendez

Date: June 28, 2026

Hi! My name is Walter Melendez, and welcome to my capstone portfolio.

I'm currently an Admissions Operations Analyst at Full Sail University and I'm working toward my Graduate Certificate in Data Science. I've always enjoyed solving problems with data, whether that's building dashboards, automating reports, or creating tools that make people's jobs easier.

This blog will document my capstone project from start to finish. My current project is a Predictive Lead-to-Student Conversion System that uses machine learning to predict whether a lead is likely to convert. Throughout the project I'll be share my research, development process, challenges, lessons learned, and the progress I make each month.

My goal is not only to complete this capstone, but also to continue building my skills in data science, machine learning, and predictive analytics while creating projects I can use in my professional portfolio.
