Predictive Lead-to-Student Conversion System

This capstone project uses machine learning to predict whether an education lead is likely to convert. The project compares Logistic Regression, Random Forest, and Support Vector Machine models.

The goal is to create a framework that could eventually be adapted to predict whether a lead becomes an active student.

Tools: Python, Pandas, Scikit-learn, Jupyter Notebook, GitHub
